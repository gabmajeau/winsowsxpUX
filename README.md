# SITE UX - WINDOWS XP COM INTERAÇÕES NOSTALGICAS

Este projeto recria uma experiência interativa inspirada no Windows clássico diretamente no navegador. O usuário encontra um desktop com ícones, menu Iniciar funcional, janelas arrastáveis e interações nostálgicas.

## Funcionalidades

- Menu Iniciar autêntico com foto de usuário editável  
- Área de trabalho com ícones que abrem janelas (vídeos, iframes, emuladores)  
- Sistema de janelas arrastáveis, com cabeçalho, ícone e botão de fechar  
- Janela para troca da foto de usuário (seleção de ícones)  
- Simulação de desligamento com overlay e diálogo central  
- Bonzi Buddy: GIF fixo ativável no canto da tela  
- Media Player especial com controles habilitados apenas para ele


## Como rodar

1. Clone ou faça download do repositório.  
2. Abra `index.html`  
4. Ou acesse: https://gabmajeau.github.io/winsowsxpUX/
