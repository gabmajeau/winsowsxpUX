# SITE UX - WINDOWS XP COM INTERAÇÕES NOSTALGICAS

Este projeto investigou como a nostalgia pode ser aplicada na cultura pop como estratégia amplificadora de um produto. A pesquisa foi realizada em duas etapas: um formulário exploratório sem foco em UX e, posteriormente, o desenvolvimento de um protótipo focado totalmente no usuário, tendo nos dois, elementos de nostalgia. A comparação entre as duas etapas evidenciou como a nostalgia, quando aplicada de forma intencional, aumenta o engajamento e fortalece a conexão emocional do usuário com o produto digital.

O objetivo principal deste projeto foi demonstrar como a nostalgia, amplamente presente na cultura pop, pode atuar como um recurso emocional poderoso para aumentar o engajamento dos usuários com um produto digital. A proposta buscou evidenciar que trazer de volta elementos marcantes de uma época — como interfaces, sons, estilos visuais e referências culturais — cria uma conexão afetiva imediata, fortalecendo o vínculo entre usuário e experiência.

Além disso, o projeto procurou analisar como a evocação de memórias positivas pode transformar simples interações em momentos de reconhecimento emocional, mostrando que a nostalgia não só atrai a atenção do público, mas também aumenta o tempo de permanência, a imersão e a sensação de valor percebido. Ao recriar um ambiente icônico de forma fiel, foi possível observar como esses elementos impactam diretamente na forma como o usuário percebe, explora e se envolve com o produto.

O desenvolvimento do projeto iniciou-se com uma pesquisa exploratória realizada por meio de um formulário no Google Forms, cujo objetivo era avaliar a reação das pessoas a conteúdos nostálgicos sem qualquer otimização de UX. Mesmo sem um design focado no usuário, os resultados mostraram um forte engajamento espontâneo, evidenciando que a nostalgia, por si só, já possui um poder significativo de atração e conexão emocional. Essa primeira etapa serviu como base para entendermos o potencial desse sentimento como recurso estratégico, além de orientar a direção do próximo experimento: criar uma experiência nostálgica profundamente estruturada, planejada e projetada com foco total no usuário. Com base nos resultados da pesquisa anterior, foi coletado que os jogos tiveram maior destaque, a partir desses resultados, desenvolvemos a segunda etapa do projeto: uma recriação interativa fiel da experiência do Windows XP, que foi um clássico para a faixa etária dos usuários anteriormente testados, a experiência foi construída para testar como a nostalgia trabalhada de maneira consciente e orientada por UX poderia amplificar ainda mais o vínculo emocional do público. Todo o ambiente foi estruturado utilizando HTML, CSS e JavaScript, garantindo compatibilidade com navegadores modernos, no momento, o desenvolvimento foi focado apenas para desktop. Cada elemento visual foi estudado cuidadosamente: os valores RGB originais do Windows XP foram replicados, os ícones foram convertidos diretamente das versões originais do sistema, os sons clássicos foram restaurados e inseridos, e até as animações de janelas e menus seguiram o comportamento autêntico da época. Além disso, foram adicionados ícones interativos com conteúdo típicos dos anos 2000 — como páginas antigas, intros de jogos e efeitos sonoros — tudo pensado para despertar lembranças e aumentar a sensação de autenticidade. Toda a interface foi construída para ser intuitiva, responsiva e fiel, colocando o usuário no centro do design e demonstrando, de forma prática, como a nostalgia aliada ao UX potencializa o engajamento e transforma a experiência em algo memorável.



## Funcionalidades

- Menu Iniciar autêntico com foto de usuário editável  
- Área de trabalho com ícones que abrem janelas (vídeos, iframes, emuladores)  
- Sistema de janelas arrastáveis, com cabeçalho, ícone e botão de fechar  
- Janela para troca da foto de usuário (seleção de ícones)  
- Simulação de desligamento com overlay e diálogo central  
- Bonzi Buddy: GIF fixo ativável no canto da tela  
- Media Player especial com controles habilitados apenas para ele
